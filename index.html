About

This project is a modern web dashboard for real-time forest fire detection using temperature and smoke sensors connected via ESP8266 WiFi module. Data is sent to Firebase and displayed live here.

  • 🌡 Temperature Sensor
  • 💨 Smoke Sensor
  • 📡 ESP8266 WiFi Module
  • 🔥 Real-time Alerts & Monitoring
  • 📊 Live Charts & Logging
  • 🎛️ System Control Panel
